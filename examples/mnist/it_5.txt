# 迭代5：修复梯度张量初始化与调试

## 概述
本次迭代针对迭代4中发现的卷积层和密集层梯度张量未初始化问题，修改了层创建函数，在创建权重和偏置张量的同时创建对应的梯度张量并初始化为零。然而，调试发现梯度张量指针在模型创建后仍为NULL，初步分析可能与动态链接库版本或函数调用链路有关。训练流程仍未完全通过，但明确了下一步调试方向。

## 完成的工作

### 1. 层初始化代码修复
- **卷积层梯度张量创建**：修改`src/layers/conv.c`中的`boat_conv_layer_create()`函数，在创建权重和偏置张量后立即创建对应的梯度张量（`grad_weight`和`grad_bias`），使用相同形状并初始化为零。
- **密集层梯度张量创建**：修改`src/layers/dense.c`中的`boat_dense_layer_create()`函数，同样在创建参数张量后创建梯度张量，确保梯度张量在层创建时即存在。
- **错误处理完善**：在梯度张量创建失败时正确释放已分配资源并返回`NULL`，保持内存安全。

### 2. 调试输出增强
- **添加详细调试日志**：在卷积层和密集层创建函数中添加`fprintf(stderr, ...)`输出，跟踪函数调用和梯度张量创建结果。
- **梯度获取函数监控**：在`boat_conv_layer_get_grad_weight()`中添加调试输出，监控梯度张量指针的实际返回值。

### 3. 编译验证
- **成功编译**：修改后的代码通过CMake/MSBuild编译，生成新的`boat.dll`动态库和`mnist_autodiff.exe`可执行文件。
- **测试通过**：卷积层前向传播测试`test_conv_forward`仍然通过，证明基础功能未受影响。

### 问题分析
1. **梯度张量未创建**：尽管代码修改正确，但运行时梯度张量指针仍为`NULL`，可能原因：
   - 动态链接库版本未更新（程序可能链接到旧版本`boat.dll`）
   - 函数调用未进入修改后的代码路径（可能存在多个实现版本）
   - 编译配置问题导致修改未生效

2. **调试输出缺失**：添加到层创建函数和梯度获取函数中的调试语句未产生输出，表明：
   - 这些函数可能未被调用（调用的是其他实现）
   - 输出被重定向或缓冲（但其他调试输出正常）
   - 函数被内联或优化（但使用了`BOAT_NOINLINE`宏）

3. **训练流程阻塞**：由于梯度张量为`NULL`，优化器无法注册参数，导致训练可能无法正确进行。

```c
// Create gradient accumulators with same shape as parameters
layer->grad_weight = boat_tensor_create(weight_shape, 4, BOAT_DTYPE_FLOAT32, BOAT_DEVICE_CPU);
if (!layer->grad_weight) {
    boat_tensor_free(layer->weight);
    boat_tensor_free(layer->bias);
    boat_free(layer);
    return NULL;
}
// Initialize gradient weight with zeros
float* grad_weight_data = (float*)boat_tensor_data(layer->grad_weight);
size_t grad_weight_elements = boat_tensor_nelements(layer->grad_weight);
memset(grad_weight_data, 0, grad_weight_elements * sizeof(float));

layer->grad_bias = boat_tensor_create(bias_shape, 1, BOAT_DTYPE_FLOAT32, BOAT_DEVICE_CPU);
if (!layer->grad_bias) {
    boat_tensor_free(layer->weight);
    boat_tensor_free(layer->bias);
    boat_tensor_free(layer->grad_weight);
    boat_free(layer);
    return NULL;
}
// Initialize gradient bias with zeros
float* grad_bias_data = (float*)boat_tensor_data(layer->grad_bias);
```

#### 密集层梯度初始化（dense.c:74-103）
```c
// Create gradient weight tensor with same shape as weight
layer->grad_weight = boat_tensor_create(weight_shape, 2, BOAT_DTYPE_FLOAT32, BOAT_DEVICE_CPU);
if (!layer->grad_weight) {
    boat_tensor_free(layer->weight);
    if (layer->bias) boat_tensor_free(layer->bias);
    boat_free(layer);
    return NULL;
}
// Initialize gradient weight with zeros
float* grad_weight_data = (float*)boat_tensor_data(layer->grad_weight);
size_t grad_weight_elements = boat_tensor_nelements(layer->grad_weight);
memset(grad_weight_data, 0, grad_weight_elements * sizeof(float));

// Create gradient bias tensor if bias is used
if (use_bias) {
    layer->grad_bias = boat_tensor_create(bias_shape, 1, BOAT_DTYPE_FLOAT32, BOAT_DEVICE_CPU);
    if (!layer->grad_bias) {
        boat_tensor_free(layer->weight);
        if (layer->bias) boat_tensor_free(layer->bias);
        boat_tensor_free(layer->grad_weight);
        boat_free(layer);
        return NULL;
    }
    // Initialize gradient bias with zeros
    float* grad_bias_data = (float*)boat_tensor_data(layer->grad_bias);
    size_t grad_bias_elements = boat_tensor_nelements(layer->grad_bias);
    memset(grad_bias_data, 0, grad_bias_elements * sizeof(float));
} else {
    layer->grad_bias = NULL;
}
```
