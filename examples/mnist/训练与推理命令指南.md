# MNIST卷积神经网络训练与推理命令指南

## 概述

本指南提供MNIST卷积神经网络的训练和推理命令说明。系统包含两个版本：基础版本（`mnist.c`）和高级自动微分版本（`mnist_autodiff.c`）。

## 1. 数据准备

数据文件已存在于 `examples/mnist/data/` 目录中，包含完整的训练集（60,000样本）和精简版（快速测试用）。如需重新生成数据：

```bash
cd D:\codes\boat\examples\mnist
python mnist_data.py
```

生成的文件：
- `train_images.bin`, `train_labels.bin` - 完整训练集
- `train_images_small.bin`, `train_labels_small.bin` - 精简训练集（快速测试）
- `test_images.bin`, `test_labels.bin` - 测试集
- `test_images_small.bin`, `test_labels_small.bin` - 精简测试集

## 2. 基础版本（mnist.c）

### 构建命令
```bash
cd D:\codes\boat\examples\mnist
# 使用CMake构建
mkdir -p build && cd build
cmake ..
cmake --build . --config Release --target mnist
```

或直接使用gcc编译：
```bash
cd D:\codes\boat\examples\mnist
gcc -I../../include -I. -L../../build/lib -o mnist mnist.c -lboat -lm -std=c99
```

### 训练与推理命令
```bash
# 使用精简数据快速测试（默认）
cd D:\codes\boat\examples\mnist
./mnist

# 使用完整数据训练
cd D:\codes\boat\examples\mnist
set USE_FULL_DATA=1
./mnist
```

基础版本特点：
- 简单CNN架构（2个卷积层、2个池化层、2个全连接层）
- 5个训练周期，学习率0.001，批量大小32
- 自动进行训练和测试集评估
- 输出每个周期的训练时间和准确率

## 3. 高级自动微分版本（mnist_autodiff.c）

### 构建命令
已有预编译可执行文件：
- `mnist_autodiff_ucb.exe` - 最新版本（包含不确定性引导探索）
- `mnist_autodiff_it22.exe` - 迭代22版本
- `mnist_autodiff_new.exe` - 较新版本

如需重新编译：
```bash
cd D:\codes\boat\examples\mnist
gcc -I../../include -I. -L../../build/lib -o mnist_autodiff mnist_autodiff.c -lboat -lm -std=c99
```

### 训练命令
```bash
cd D:\codes\boat\examples\mnist
# 使用最新版本
./mnist_autodiff_ucb.exe

# 或指定版本
./mnist_autodiff_it22.exe
```

高级版本特点：
- 完整的自动微分和反向传播
- 自适应学习率调度
- 梯度裁剪和监控
- 强化学习超参数调优（RL代理）
- 训练阶段感知的自适应动作空间
- 基于不确定性的动作选择（最新版本）
- 详细的训练统计和监控日志
- 自动分割训练/验证集（5000个验证样本）

### 推理命令
高级版本支持加载已保存的模型检查点：
```bash
cd D:\codes\boat\examples\mnist
./mnist_autodiff_ucb.exe --load model_checkpoint.bin
```

注意：当前代码中检查点保存功能需要额外实现，`--load` 参数为预留接口。

## 4. 数据路径说明

### 基础版本数据路径
- 默认使用 `examples/mnist/data/` 目录中的二进制文件
- 精简数据：`*_small.bin` 文件
- 完整数据：通过 `USE_FULL_DATA` 环境变量切换

### 高级版本数据路径
- 硬编码路径：`D:/huggingface/student/MNIST/MNIST/MNIST/raw/`
- 需要原始的IDX格式文件：
  - `train-images-idx3-ubyte`
  - `train-labels-idx1-ubyte`

  - `t10k-labels-idx1-ubyte`

如需修改数据路径，需要编辑 `mnist_autodiff.c` 第2281行：


## 7. 故障排除

### 常见问题
1. **缺少Boat库**：确保 `libboat.so` 或 `boat.dll` 在库路径中
2. **数据文件找不到**：运行 `python mnist_data.py` 重新生成
3. **IDX文件找不到**（高级版本）：将MNIST IDX文件复制到指定目录或修改源代码中的路径
4. **内存不足**：使用精简数据集（默认）或减少批量大小

### 调试建议
```bash
# 启用详细输出
set BOAT_DEBUG=1
./mnist_autodiff_ucb.exe

# 检查库依赖（Linux）
ldd mnist_autodiff_ucb.exe

# 检查库依赖（Windows）
dumpbin /dependents mnist_autodiff_ucb.exe
```


对于完整训练和高级功能，使用高级版本：

两个版本都包含完整的训练和推理流程，训练结束后自动在测试集上评估并输出准确率。高级版本提供了更丰富的监控和自适应调优功能，适合研究和使用强化学习进行超参数优化。
