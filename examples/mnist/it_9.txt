### 4. 现有测试增强

- **增强诊断**：当梯度检查失败时，除了显示不匹配元素外，还输出误差统计信息，帮助定位数值问题。

## 技术细节

### 双精度数值梯度实现
```c
static double compute_numerical_gradient_element_fp64(boat_conv_layer_t* layer,
                                                     boat_tensor_t* input,
                                                     boat_tensor_t* param,
                                                     size_t idx,
                                                     double epsilon) {
    // 保存原始值（float32）
    float* data = (float*)boat_tensor_data(param);
    float original = data[idx];

    // 正向扰动计算（双精度累加）
    data[idx] = original + (float)epsilon;
    boat_tensor_t* output_plus = boat_conv_layer_forward(layer, input);
    double loss_plus = 0.0;
    if (output_plus) {
        float* out_data = (float*)boat_tensor_data(output_plus);
        size_t n = boat_tensor_nelements(output_plus);
        for (size_t i = 0; i < n; i++) {
            loss_plus += (double)out_data[i];  // 双精度累加
        }
        boat_tensor_unref(output_plus);
    }
    // ... 类似处理负向扰动
    return (loss_plus - loss_minus) / (2.0 * epsilon);
}
```

### 误差统计收集
在 `test_parameter_gradient()` 函数中，对每个测试元素计算：
1. 相对误差：`|analytical - numerical| / (|analytical| + |numerical|)`
2. 绝对误差：`|analytical - numerical|`
3. 更新最大值和累加平均值

### 配置管理
通过两个全局标志控制行为：
```c
static bool enable_gradient_analysis = false;        // 启用详细误差统计
static bool use_double_precision_numerical = false;  // 使用双精度数值梯度
```
在 `main()` 函数中设置为 `true` 以启用分析功能。
