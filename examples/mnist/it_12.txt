# 迭代12：模型检查点实现与训练恢复支持

## 概述
本次迭代基于迭代11的训练性能优化基础，专注于模型检查点功能的实现。主要完成了MNIST模型权重保存/加载、最佳模型自动保存、训练中断恢复支持，为模型持久化和生产部署奠定了基础。

## 完成的工作

### 1. 模型检查点API实现
- **张量序列化格式**：设计了简单的二进制格式，支持任意形状和数据类型张量的保存与加载。
- **辅助函数**：实现了`save_tensor_to_file()`和`load_tensor_from_file()`静态辅助函数，处理张量元数据（维度、形状、数据类型）和数据的序列化。
- **模型保存/加载**：实现了`save_mnist_model()`和`load_mnist_model()`函数，保存和加载MNIST模型的所有可训练参数（4个权重张量 + 4个偏置张量）。
- **文件格式**：包含魔数（0x4D4E4953 'MNIS'）和版本号，确保文件兼容性。

### 2. 训练循环集成
- **最佳模型自动保存**：在训练循环中跟踪最佳验证准确率，当准确率提升时自动保存检查点。
- **定期检查点**：每2个epoch保存一次最新检查点（`mnist_checkpoint_latest.boat`）。
- **最终模型保存**：训练结束后保存最终模型（`mnist_final.boat`）。
- **训练信息输出**：显示最佳模型信息（epoch和验证准确率）。

### 3. 命令行参数支持
- **检查点加载**：添加`--load <file>`参数，支持从检查点恢复训练。
- **帮助信息**：添加`--help`参数显示使用说明。
- **向后兼容**：保持无参数运行的默认行为。

### 4. 错误处理与健壮性
- **文件操作安全**：所有文件操作包含错误检查，提供有意义的错误信息。
- **内存管理**：正确管理张量引用计数，避免内存泄漏。
- **版本验证**：加载时验证魔数和版本号，防止加载不兼容文件。

## 技术细节

### 检查点文件格式
```
文件格式：
+------------------+
| 魔数 (4字节)     | 0x4D4E4953 ('MNIS')
+------------------+
| 版本号 (4字节)   | 1
+------------------+
| 张量1元数据      | (见张量格式)
+------------------+
| 张量1数据        |
+------------------+
| ...              | 重复8次（4权重+4偏置）
+------------------+

张量格式：
+------------------+
| 是否为NULL (4B)  | 1=Null, 0=有数据
+------------------+
| 维度数 (4B)      | ndim
+------------------+
| 形状[0] (4B)     |
| ...              | ndim个形状值
| 形状[ndim-1] (4B)|
+------------------+
| 数据类型 (4B)    | boat_dtype_t枚举值
+------------------+
| 数据 (变长)      | nbytes = 总元素数 * 元素大小
+------------------+
```

### 模型参数序列化顺序
1. Conv1 权重 (形状: [32, 1, 3, 3])
2. Conv1 偏置 (形状: [32])
3. Conv2 权重 (形状: [64, 32, 3, 3])
4. Conv2 偏置 (形状: [64])
5. FC1 权重 (形状: [128, 3136])
6. FC1 偏置 (形状: [128])
7. FC2 权重 (形状: [10, 128])
8. FC2 偏置 (形状: [10])

```c
// 模型保存
bool save_mnist_model(const mnist_model_t* model, const char* filename) {
    FILE* f = fopen(filename, "wb");
    uint32_t magic = 0x4D4E4953; // 'MNIS'
    uint32_t version = 1;
    fwrite(&magic, sizeof(uint32_t), 1, f);
    fwrite(&version, sizeof(uint32_t), 1, f);

    // 保存所有参数张量
    save_tensor_to_file(f, boat_conv_layer_get_weight(model->conv1));
    save_tensor_to_file(f, boat_conv_layer_get_bias(model->conv1));
    // ... 其他参数
    fclose(f);
    return true;
}
```
