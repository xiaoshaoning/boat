### 📊 实现正确性验证

#### 1. 数学公式验证

卷积层梯度公式的理论推导与实现完全一致：

**前向传播**：
```
output[b,oc,oh,ow] = Σ_{ic,kh,kw} input[b,ic,ih,iw] × weight[oc,ic,kh,kw] + bias[oc]
where ih = oh×stride - padding + kh, iw = ow×stride - padding + kw
```

**反向传播**（已验证）：
```
∂L/∂input[b,ic,ih,iw] = Σ_{oc,kh,kw} ∂L/∂output[b,oc,oh,ow] × weight[oc,ic,kh,kw]
∂L/∂weight[oc,ic,kh,kw] = Σ_{b,oh,ow} ∂L/∂output[b,oc,oh,ow] × input[b,ic,ih,iw]
∂L/∂bias[oc] = Σ_{b,oh,ow} ∂L/∂output[b,oc,oh,ow]
```

#### 2. 边界条件测试

**填充处理**：
- 输入索引自动跳过越界位置（`ih < 0 || ih >= height`）
- 与PyTorch Conv2d实现行为一致

**步长支持**：
- 支持任意正整数步长
- 输出形状计算正确：`height_out = (height + 2×padding - kernel_size) / stride + 1`

**数据类型**：
- 当前仅支持FLOAT32（符合Phase 1目标）
- 类型检查确保安全

#### 3. 内存与性能特征

**内存使用**：
- 梯度张量独立分配，避免原地操作冲突
- 前向传播缓存输入张量（引用计数管理）
- 峰值内存使用可控

**算法复杂度**：
- 输入梯度：O(B×C_out×C_in×H_out×W_out×K_h×K_w)
- 权重梯度：O(B×C_out×C_in×H_out×W_out×K_h×K_w)
- 偏置梯度：O(B×C_out×H_out×W_out)

**当前限制**（计划在后续阶段优化）：
- 朴素7层嵌套循环实现
- 未使用SIMD指令集优化
- 仅支持CPU计算

```c
boat_conv_layer_t* conv1 = boat_conv_layer_create(3, 32, 3, 1, 1);

    // 前向传播

    boat_tensor_t* out2 = boat_conv_layer_forward(conv2, out1);

    // 计算损失和梯度

    boat_conv_layer_backward(conv1, grad1);

    // 参数更新
    boat_conv_layer_update(conv1, 0.001f);
    boat_conv_layer_update(conv2, 0.001f);
}
```
