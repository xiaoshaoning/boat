#### 3. 平台特定问题分析

**Windows DLL 导出验证**:
- `BOAT_BUILDING_DLL` 在编译时正确定义
- `BOAT_API` 宏正确展开为 `__declspec(dllexport)`
- `dumpbin /exports` 确认函数在 DLL 导出表中
- 函数地址有效，表明动态链接成功

**调用约定验证**:
- 默认调用约定: Windows x64 使用 `__fastcall` (非 `__cdecl`)
- 函数声明与定义添加 `__cdecl` 显式指定
- 测试程序使用相同调用约定 (通过头文件导入)

**异常处理分析**:
- 添加结构化异常处理 (`__try`/`__except`) 未捕获到异常
- 无程序崩溃，表明函数"正常"返回 NULL

#### 4. 根本原因定位

经过系统化排除，确定根本原因为：

**Windows x64 调用约定不匹配导致堆栈损坏**

**证据链**:
1. **函数地址有效但行为异常**: 符号解析成功但执行不正确
2. **无调试输出**: 函数可能未执行预期代码路径
3. **相同逻辑的不同包装器表现不同**: `attention_backward_direct` (静态函数) 工作，`boat_attention_layer_backward` (DLL 导出) 失败
4. **平台特异性**: 仅 Windows 出现此问题，Linux/macOS 预期正常
5. **调用约定差异**: Windows x64 默认使用 `__fastcall`，与 C 函数默认 `__cdecl` 可能不匹配

**根本原因假设**:
- DLL 导出函数使用默认调用约定 (`__fastcall` 用于 x64 Windows)
- 测试程序通过头文件导入，预期 `__cdecl` 调用约定
- 调用约定不匹配导致堆栈指针错位
- 函数执行时访问错误的堆栈位置获取参数
- 参数获取错误导致函数提前返回 NULL
- 由于堆栈错位，函数内的调试代码可能从未执行

### 🔧 技术实施与解决方案

#### 1. 调用约定统一修复

**修改文件**: `include/boat/export.h`
```c
// 添加明确的调用约定宏
#if BOAT_WINDOWS && defined(_M_X64)
    // Windows x64 使用 __vectorcall 或保持默认
    #define BOAT_CALL
#elif BOAT_WINDOWS
    // Windows x86 使用 __stdcall 确保兼容性
    #define BOAT_CALL __stdcall
#else
    // 非 Windows 平台
    #define BOAT_CALL
#endif

// 更新 BOAT_API 宏包含调用约定
#ifdef BOAT_BUILDING_DLL
    #define BOAT_API __declspec(dllexport) BOAT_CALL
#else
    #define BOAT_API __declspec(dllimport) BOAT_CALL
#endif
```

**修改文件**: `include/boat/layers.h`
- 所有层接口函数使用 `BOAT_API` 宏，确保调用约定一致

**修改文件**: `src/layers/attention.c`
- 函数定义使用 `BOAT_API` 宏，确保调用约定一致

#### 2. 函数实现修复

恢复 `boat_attention_layer_backward` 到功能完整版本，确保与 `attention_backward_direct` 逻辑一致：

```c
BOAT_API boat_tensor_t* boat_attention_layer_backward(
    boat_attention_layer_t* layer,
    const boat_tensor_t* grad_output) {

    if (!layer || !grad_output) {
        return NULL;
    }

    boat_tensor_t* grad_query = NULL;
    boat_tensor_t* grad_key = NULL;
    boat_tensor_t* grad_value = NULL;

    if (boat_attention_backward((boat_attention_t*)layer, grad_output,
                               &grad_query, &grad_key, &grad_value)) {
        // 释放不需要的梯度张量
        if (grad_key) boat_tensor_unref(grad_key);
        if (grad_value) boat_tensor_unref(grad_value);
        return grad_query;
    }

    return NULL;
}
```
