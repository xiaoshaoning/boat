#### 1. 缩放点积注意力完整实现
- **问题诊断**: `boat_scaled_dot_product_attention` 函数为占位符实现，仅复制输入数据
- **修复方案**: 在 `src/layers/attention.c` 中实现完整的缩放点积注意力算法
- **实现细节**:
  ```c
  // 支持4D张量形状: [batch, num_heads, seq_len, head_size]
  // 计算流程:
  // 1. 计算注意力分数: scores = Q @ K^T * scale_factor
  // 2. 应用因果掩码 (causal_mask): 屏蔽未来位置
  // 3. Softmax归一化: scores = softmax(scores, dim=-1)
  // 4. 注意力输出: output = scores @ V
  // 5. 支持批量处理和多头并行计算
  ```
- **关键特性**:
  - 完整的浮点矩阵乘法循环实现
  - 因果掩码支持（上三角矩阵掩码）
  - 数值稳定的Softmax实现（减去最大值）
  - 内存高效：每头使用临时分数矩阵
  - 错误处理：形状验证、内存分配检查
- **技术参数**:
  - 支持数据类型: `BOAT_DTYPE_FLOAT32`
  - 输入形状: 4D张量 [batch, num_heads, seq_len, head_size]
  - 输出形状: 与value相同
  - 计算复杂度: O(batch×num_heads×seq_len²×head_size)

#### 1. src/layers/attention.c 缩放点积注意力实现
```c
// 核心计算循环（简化）
for (int64_t b = 0; b < batch; b++) {
    for (int64_t h = 0; h < num_heads; h++) {
        // 计算 Q @ K^T * scale
        for (int64_t i = 0; i < seq_len; i++) {
            for (int64_t j = 0; j < seq_len; j++) {
                float sum = 0.0f;
                for (int64_t d = 0; d < head_size; d++) {
                    sum += q_head[i*head_size+d] * k_head[j*head_size+d];
                }
                scores[i*seq_len+j] = sum * scale;
            }
        }

        // 因果掩码
        if (causal_mask) {
            for (int64_t i = 0; i < seq_len; i++) {
                for (int64_t j = i+1; j < seq_len; j++) {
                    scores[i*seq_len+j] = -1e9f;
                }
            }
        }

        // Softmax
        for (int64_t i = 0; i < seq_len; i++) {
            float max_val = scores[i*seq_len];
            // ... 标准化实现
        }

        // 注意力输出: scores @ V
        for (int64_t i = 0; i < seq_len; i++) {
            for (int64_t d = 0; d < head_size; d++) {
                float sum = 0.0f;
                for (int64_t j = 0; j < seq_len; j++) {
                    sum += scores[i*seq_len+j] * v_head[j*head_size+d];
                }
                out_head[i*head_size+d] = sum;
            }
        }
    }
}
```
