- **实现细节**：
  - 基于 `clock()` 计时和每个epoch处理的样本总数

- **输出集成**：在epoch进度输出中新增吞吐量字段，格式为：`throughput=XXXX samples/s`

### 吞吐量计算实现
```c
// 计算本epoch处理的样本数
int epoch_total = ...;  // 实际处理的样本总数
double epoch_time = ...; // epoch耗时（秒）

// 计算吞吐量（防止除零）
double samples_per_second = epoch_time > 1e-6 ?
                           (double)epoch_total / epoch_time : 0.0;

// 集成到输出中
printf("Epoch %d/%d: time=%.2fs, loss=%.4f, accuracy=%.2f%%, throughput=%.0f samples/s\n",
       epoch + 1, epochs, epoch_time, avg_loss, epoch_accuracy * 100.0f, samples_per_second);
```
