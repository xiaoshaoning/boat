## 卷积层反向传播实现报告

基于 `docs/it_safetensors_48.txt` 中的卷积层前向传播实现，我们完成了卷积神经网络层的反向传播（梯度计算）功能。本报告记录实现细节、技术设计、测试验证结果以及后续优化方向。

### 🎯 实现目标

为Boat深度学习框架的卷积层（Conv2D）添加完整的反向传播功能，支持：
- 输入张量的梯度计算（∂loss/∂input）
- 权重张量的梯度计算（∂loss/∂weight）
- 偏置张量的梯度计算（∂loss/∂bias，如果启用）
- 梯度累积和参数更新
- 完整的错误检查和边界处理

### 🔧 技术实现

#### 1. 核心算法实现

**文件位置**: `src/layers/conv.c`
**实现函数**: `boat_conv_layer_backward()`, `boat_conv_layer_update()`

**反向传播算法设计**:
- **输入梯度**: ∂L/∂input = 卷积转置(∂L/∂output, weight)
  - 使用权重张量和输出梯度计算输入梯度
  - 实现7层嵌套循环，遍历batch、输出通道、输入通道、核位置、输出位置
- **权重梯度**: ∂L/∂weight = 卷积(∂L/∂output, input)
  - 使用输入张量和输出梯度计算权重梯度
  - 相同的7层嵌套循环结构，但累加方向不同
- **偏置梯度**: ∂L/∂bias = sum(∂L/∂output, axis=[batch, height, width])
  - 对输出梯度在批次、高度、宽度维度上求和
  - 仅当启用偏置时计算

**数学公式**:
```
// 前向传播
output[b,oc,oh,ow] = sum_{ic,kh,kw} input[b,ic,ih,iw] × weight[oc,ic,kh,kw] + bias[oc]
where ih = oh×stride - padding + kh, iw = ow×stride - padding + kw

// 反向传播
∂L/∂input[b,ic,ih,iw] = sum_{oc,kh,kw} ∂L/∂output[b,oc,oh,ow] × weight[oc,ic,kh,kw]
∂L/∂weight[oc,ic,kh,kw] = sum_{b,oh,ow} ∂L/∂output[b,oc,oh,ow] × input[b,ic,ih,iw]
∂L/∂bias[oc] = sum_{b,oh,ow} ∂L/∂output[b,oc,oh,ow]
```

**缓存机制**:
- 前向传播时缓存输入张量和输入/输出形状
- 反向传播时使用缓存数据计算梯度
- 避免重复计算形状和内存分配

#### 2. 参数更新实现

**更新函数**: `boat_conv_layer_update()`
**优化算法**: 随机梯度下降（SGD）
**更新规则**:
```
weight = weight - learning_rate × grad_weight
bias = bias - learning_rate × grad_bias
```

**梯度管理**:
- 梯度张量存储在层结构中（grad_weight, grad_bias）
- 支持梯度累积（多次反向传播后更新）
- 提供梯度获取接口（`boat_conv_layer_get_grad_weight()`等）

#### 3. 错误处理与验证

**输入验证**:
1. 层指针和梯度输出张量非空检查
2. 缓存输入存在性验证（确保前向传播已调用）
3. 梯度输出形状与缓存输出形状匹配检查
4. 数据类型一致性验证（仅支持FLOAT32）

**错误消息**:
- "Error: conv backward: NULL input"
- "Error: conv backward: no cached input (forward not called or cache cleared)"
- "Error: conv backward: grad_output shape doesn't match cached output shape"
- "Error: conv backward: failed to compute input/weight gradient"

### 📊 性能特征

#### 1. 算法复杂度
- **输入梯度计算**: O(B × C_out × C_in × H_out × W_out × K_h × K_w)
- **权重梯度计算**: O(B × C_out × C_in × H_out × W_out × K_h × K_w)
- **偏置梯度计算**: O(B × C_out × H_out × W_out)
- **总复杂度**: 与前向传播相同量级

#### 2. 内存使用
- **梯度存储**: 需要额外存储grad_weight和grad_bias张量
- **缓存**: 存储输入张量引用和形状信息
- **峰值内存**: 前向传播输出 + 输入梯度 + 权重梯度 + 偏置梯度

#### 3. 当前限制
- 仅支持FLOAT32数据类型
- 朴素实现，未使用SIMD或并行优化
- 不支持分组卷积、空洞卷积等高级特性
- 仅支持SGD优化器更新规则

#### 4. 优化潜力
- **算法优化**: IM2COL转换，Winograd算法
- **硬件加速**: SIMD向量化，多线程并行
- **内存优化**: 梯度计算原地操作，内存复用
- **数值优化**: 混合精度训练，梯度裁剪

### 📋 部署与使用

#### 1. 使用示例
```c
#include <boat/layers.h>

// 创建卷积层
boat_conv_layer_t* conv = boat_conv_layer_create(3, 16, 3, 1, 1);

// 前向传播
boat_tensor_t* output = boat_conv_layer_forward(conv, input);

// 计算损失梯度（假设已有）
boat_tensor_t* grad_output = ...;

// 反向传播（计算梯度）
boat_tensor_t* grad_input = boat_conv_layer_backward(conv, grad_output);

// 参数更新
boat_conv_layer_update(conv, 0.001f);

// 获取梯度（可选）
boat_tensor_t* grad_weight = boat_conv_layer_get_grad_weight(conv);
boat_tensor_t* grad_bias = boat_conv_layer_get_grad_bias(conv);

// 清理
boat_tensor_unref(output);
boat_tensor_unref(grad_input);
boat_conv_layer_free(conv);
```

#### 2. 参数说明
- `learning_rate`: 学习率，控制参数更新步长
- `grad_output`: 损失函数对层输出的梯度，形状与层输出相同
- 返回的`grad_input`: 损失函数对层输入的梯度，形状与层输入相同

#### 3. 注意事项
- 必须在调用`boat_conv_layer_forward()`后调用`boat_conv_layer_backward()`
- 梯度计算使用前向传播时缓存的输入数据
- 参数更新使用累积的梯度，支持多次反向传播后一次更新
- 当前仅支持FLOAT32数据类型和SGD优化器
