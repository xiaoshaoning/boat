基于 `it_safetensors_26.txt` 中的下一步计划，我们成功实现了注意力层的完整梯度计算算法，包括最终线性投影梯度、缓存系统扩展和参数更新集成，为端到端训练提供了完整的自动微分支持。

### 🔧 完成的工作

#### 1. 缓存系统扩展
- **新增缓存字段**: 在 `boat_attention_t` 结构体中添加 `cache_attention_output` 字段，存储最终线性投影前的注意力输出张量
- **内存管理增强**: 在 `boat_attention_free` 和 `boat_attention_forward` 中正确管理新缓存张量的生命周期
- **引用计数**: 所有缓存张量增加引用计数确保后向传播时数据可用

#### 2. 最终线性投影梯度计算
- **梯度算法实现**: 实现 `linear_projection_backward` 函数支持最终投影 `W_o` 和 `b_o` 的梯度计算
- **梯度存储**: 计算得到的 `grad_weight_o` 和 `grad_bias_o` 存储在注意力层结构体中
- **错误处理**: 完善的错误检查和资源清理，确保内存安全

#### 3. 后向传播完整实现
- **梯度链式传播**: 实现完整的梯度传播链：`grad_output` → `grad_attention_output` → `grad_q_proj/grad_k_proj/grad_v_proj` → `grad_q/grad_k/grad_v`
- **注意力机制梯度**: 完善 `attention_backward` 函数，实现缩放点积注意力的梯度计算
- **线性投影梯度**: 三个输入投影（Q、K、V）的梯度计算和累积

#### 4. 参数更新完整实现
- **权重更新**: 取消注释 `weight_o` 和 `bias_o` 的更新代码，实现所有可训练参数的 SGD 更新
- **梯度累积**: 支持多次后向传播的梯度累积，为小批量训练做准备
- **学习率支持**: 完整的 SGD 更新公式：`weight = weight - learning_rate * gradient`

#### 5. 编译错误修复

```c
struct boat_attention_t {
    // ... 原有配置和权重字段 ...

    // 缓存系统（后向传播用）
    boat_tensor_t* cache_query;           // 原始查询输入
    boat_tensor_t* cache_key;             // 原始键输入
    boat_tensor_t* cache_value;           // 原始值输入
    boat_tensor_t* cache_q_proj;          // 查询投影后张量
    boat_tensor_t* cache_k_proj;          // 键投影后张量
    boat_tensor_t* cache_v_proj;          // 值投影后张量
    boat_tensor_t* cache_attention_weights; // 注意力权重 [B,H,S,S]
    boat_tensor_t* cache_attention_output;  // 新增：最终投影前的注意力输出
};
```

#### 2. 后向传播梯度计算流程
```c
boat_tensor_t* boat_attention_backward(boat_attention_t* attention,
                                        const boat_tensor_t* grad_output) {
    // 步骤1：最终投影梯度（W_o, b_o）
    linear_projection_backward(cache_attention_output, weight_o, bias_o,
                               grad_output, &grad_attention_output,
                               &grad_weight_o, &grad_bias_o);

    // 步骤2：注意力机制梯度（Q_proj, K_proj, V_proj）
    attention_backward(cache_q_proj, cache_k_proj, cache_v_proj,
                       cache_attention_weights, grad_attention_output,
                       &grad_q_proj, &grad_k_proj, &grad_v_proj);

    // 步骤3：输入投影梯度（W_q, W_k, W_v）
    linear_projection_backward(cache_query, weight_q, bias_q,
                               grad_q_proj, &grad_q, &grad_weight_q, &grad_bias_q);
    // ... 类似处理 K 和 V ...

    // 步骤4：存储所有梯度
    attention->grad_weight_o = grad_weight_o;
    attention->grad_bias_o = grad_bias_o;
    // ... 存储其他梯度 ...

    return grad_q;  // 返回查询输入的梯度
}
```

#### 3. 线性投影梯度计算算法
```c
// 支持 3D 张量 [batch, seq_len, hidden] 的梯度计算
static bool linear_projection_backward(const boat_tensor_t* input,
                                       const boat_tensor_t* weight,
                                       const boat_tensor_t* bias,
                                       const boat_tensor_t* grad_output,
                                       boat_tensor_t** grad_input,
                                       boat_tensor_t** grad_weight,
                                       boat_tensor_t** grad_bias) {
    // 计算 grad_input = grad_output @ weight^T
    // 计算 grad_weight = input^T @ grad_output（批处理和序列维度求和）
    // 计算 grad_bias = sum(grad_output, axis=(0,1))
    // 支持 2D 和 3D 张量形状
}
```

#### 4. 注意力机制梯度计算
```c
// 缩放点积注意力梯度（支持缓存注意力权重）
static bool attention_backward(const boat_tensor_t* query,
                               const boat_tensor_t* key,
                               const boat_tensor_t* value,
                               const boat_tensor_t* attention_weights,
                               const boat_tensor_t* grad_output,
                               boat_tensor_t** grad_query,
                               boat_tensor_t** grad_key,
                               boat_tensor_t** grad_value) {
    // 1. 计算 dV = A^T @ dO
    // 2. 计算 dA = dO @ V^T
    // 3. 计算 softmax 梯度：dS = A * (dA - sum(dA * A, dim=-1, keepdim=True))
    // 4. 应用缩放因子：dS_scaled = dS * scale
    // 5. 计算 dQ = dS_scaled @ K
    // 6. 计算 dK = dS_scaled^T @ Q
}
```

#### 5. 参数更新实现
```c
void boat_attention_update(boat_attention_t* attention, float learning_rate) {
    // 更新所有四个权重矩阵
    if (attention->grad_weight_q && attention->weight_q) {
        boat_tensor_t* scaled_grad = boat_mul_scalar(attention->grad_weight_q, learning_rate);
        boat_sub_(attention->weight_q, scaled_grad);
        boat_tensor_unref(scaled_grad);
    }
    // ... 更新 weight_k, weight_v, weight_o ...

    // 更新所有偏置向量
    if (attention->grad_bias_q && attention->bias_q) {
        boat_tensor_t* scaled_grad = boat_mul_scalar(attention->grad_bias_q, learning_rate);
        boat_sub_(attention->bias_q, scaled_grad);
        boat_tensor_unref(scaled_grad);
    }
    // ... 更新 bias_k, bias_v, bias_o ...
}
```
