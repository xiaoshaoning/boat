### 🔍 梯度错误深度调试

#### 1. 数学公式验证
- **注意力机制反向传播公式**: 重新推导标准注意力机制梯度公式
  - 前向: S = QKᵀ/√dₖ → A = softmax(S) → O = AV
  - 反向: dV = AᵀdO, dA = dOVᵀ, dS = softmax_backward(dA, A), dQ = dS·K/√dₖ, dK = dSᵀ·Q/√dₖ
- **代码实现对照**: 验证 `attention_backward` 函数与数学公式一致性
  - ✅ dV计算正确: `grad_value_local = boat_matmul(attn_t, grad_output_4d)`
  - ✅ dA计算正确: `grad_attn = boat_matmul(grad_output_4d, value_t)`
  - ✅ softmax梯度公式正确: `dS = A * (dA - sum(dA * A, dim=-1))`
  - ✅ dQ计算正确: `grad_query_local = boat_matmul(grad_scores_scaled, key)`
  - 🔄 dK计算公式修正: 原实现 `dK = dS_scaled^T @ Q` 改为 `dK = (Q^T @ dS_scaled)^T`
  - **结论**: 核心数学公式基本正确，dK计算存在转置问题但影响有限

#### 2. softmax梯度实现改进
- **原始实现**: 使用单精度浮点数累加，可能存在累积误差
- **改进措施**: 将累加器升级为双精度，减少舍入误差
  ```c
  // 修改前: float sum = 0.0f;
  // 修改后: double sum = 0.0;
  for (int64_t j = 0; j < seq_len; j++) {
      sum += (double)dA_data[idx] * (double)A_data[idx];
  }
  ```
- **验证结果**: 数值稳定性略有改善，但梯度检查差异未显著减少

#### 3. 数值梯度验证实验
- **epsilon值调整实验**:
  | epsilon值 | 通过参数数量 | 典型差异范围 | 观察结果 |
  |-----------|--------------|--------------|----------|
  | 1e-4 (默认) | 2/8 (b_o, W_o) | 0.001-0.005 | 有限差分误差相对稳定 |
  | 1e-5 | 0/8 | 0.01-0.05 | 误差显著增大，数值梯度不准确 |
  | 1e-3 | 2/8 | 0.0005-0.002 | 误差减小但仍有失败 |

- **容差调整实验**:
  | 相对容差 | 通过参数数量 | 观察结果 |
  |----------|--------------|----------|
  | 0.1% (默认) | 2/8 | 多数参数失败 |
  | 1% | 4/8 | W_o, b_o通过，其他部分失败 |
  | 2% | 5/8 | 失败数减少但仍存在 |

- **关键发现**:
  1. 有限差分误差对epsilon值敏感，1e-4为合理平衡点
  2. 解析梯度与数值梯度符号一致，数值差异在0.1%-5%范围内
  3. 最终投影层(W_o, b_o)梯度检查通过，表明线性投影反向传播正确
