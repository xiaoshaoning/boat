### 🔧 技术实现细节

#### 1. 构建器系统架构
```
原始状态（仅支持密集层）：
- complete_builder(): 只处理 "dense" 类型
- set_builder_bias(): 只更新密集层偏置
- 层归一化：直接创建层，未使用构建器

扩展后状态（支持密集层和层归一化）：
- complete_builder(): 支持 "dense" 和 "layer_norm" 类型
- set_builder_bias(): 支持密集层和层归一化层更新
- 层归一化：使用构建器系统，权重和偏置正确关联
```

#### 2. 层归一化构建器处理流程
```c
// 1. 检测层归一化参数
if (strstr(layer_name, "layer_norm") != NULL || strstr(layer_name, "ln") != NULL) {
    // 2. 获取基础层名称（如 "layer_norm"）
    char* base_name = get_base_layer_name(layer_name);

    // 3. 创建或查找构建器
    hf_layer_builder_t* builder = find_or_create_builder(config, base_name, "layer_norm");

    // 4. 根据张量类型设置权重或偏置
    if (strstr(layer_name, "weight") != NULL && ndim == 1) {
        set_builder_weight(builder, weight);  // 设置gamma参数
    } else if (strstr(layer_name, "bias") != NULL && ndim == 1) {
        set_builder_bias(builder, weight);    // 设置beta参数
    }

    // 5. 如果构建器已有权重，完成层创建
    if (builder->has_weight) {
        if (builder->layer) {
            // 层已存在（偏置更新）
            return (boat_layer_t*)1;  // 特殊标记
        } else {
            // 创建新层
            return complete_builder(builder, config);
        }
    }
}
```

#### 3. 层归一化配置传递
```c
// complete_builder() 中处理层归一化配置
boat_layernorm_config_t ln_config = {
    .normalized_shape = normalized_shape,  // 从权重张量形状获取
    .eps = config ? config->layer_norm_eps : 1e-5f,  // 从模型配置获取
    .elementwise_affine = true,
    .use_bias = builder->has_bias
};
```
