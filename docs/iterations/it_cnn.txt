Core Issue: Symbol Export Configuration
---------------------------------------

### Problem Description
The Boat library (`boat.dll`) compiled in `build_test_pytorch3/Release/` does not properly
export C function symbols, causing linking failures for test programs.

### Error Manifestations
1. **Static Linking Failures**:
   ```
   undefined reference to `__imp_boat_pytorch_load'
   undefined reference to `boat_tensor_create'
   undefined reference to `boat_model_forward'
   ```
   (Total 16 undefined references)

2. **Dynamic Loading Failures**:
   - Error 126: The specified module could not be found
   - boat.dll has dependencies on PyTorch DLLs (torch_cpu.dll, c10.dll, etc.)
   - Even with all DLLs present, loading fails due to potential missing Visual C++ runtime

### Root Cause
The export macro system (`include/boat/export.h`) was implemented in Phase 2.8, but the
existing `boat.dll` was compiled before these changes. The library needs to be rebuilt
with proper export definitions.

### 1. Static Linking (Failed)

**Result**: Error 126 - DLL loading fails even with all dependencies present.

Recommended Solution: Rebuild Boat Library
------------------------------------------

### Step 1: Clean and Reconfigure Build
```bash
cd D:\codes\boat
rm -rf build_mnist_test
mkdir build_mnist_test
cd build_mnist_test
```

### Step 2: Configure with CMake
**Using CMake-GUI (Recommended)**:
1. Source path: `D:/codes/boat`
2. Build path: `D:/codes/boat/build_mnist_test`
3. Configure options:
   - `BOAT_WITH_PYTORCH=ON`
   - `BOAT_PYTORCH_PATH=D:/codes/boat/external/libtorch-python-cpu`
   - `BOAT_WITH_TESTS=ON`
4. Configure → Generate → Open boat.sln → Build Release

**Using Command Line**:
```cmd
# In Visual Studio Developer Command Prompt
cd D:\codes\boat
mkdir build_mnist_test
cd build_mnist_test
cmake -DBOAT_WITH_PYTORCH=ON ^
      -DBOAT_PYTORCH_PATH="D:/codes/boat/external/libtorch-python-cpu" ^
      -DBOAT_WITH_TESTS=ON ..
cmake --build . --config Release
```

### Step 3: Compile Test Program
```bash
cd D:\codes\boat\external\mnist-cnn-digit-classifier
gcc -std=c11 -Wall -I../../include -L../../build_mnist_test/Release \
    test_boat_load.c -lboat -lm -o test_mnist.exe
```

### Step 4: Prepare Dependencies and Run
```bash
# Copy required DLLs
cp ../../build_mnist_test/Release/boat.dll .
cp ../../external/libtorch-python-cpu/lib/*.dll .

# Run test
./test_mnist.exe
```

