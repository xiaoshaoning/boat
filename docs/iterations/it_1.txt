Boat Deep Learning Framework - Initial Project Structure

Overview:
A lightweight, high-performance deep learning framework written in C with eventual CUDA support.
Designed for inference, training, and fine-tuning of neural networks with support for common model formats.

Key Features:
1. Pure C implementation with minimal dependencies
2. Automatic differentiation with computational graph
3. Support for multiple data types including low-bit networks (1-bit, 2-bit, 4-bit, 8-bit)
4. Graph-based model processing
5. CUDA support planned for future

Data Types Supported:
- Standard floating point: FP64 (double), FP32 (float), FP16 (half)
- Custom floating point: FP8, FP4
- Integer types: INT64, INT32, UINT8
- Low-bit quantization: BITS2 (2-bit), BITS1 (1-bit binary networks)
- Boolean: BOOL

Core Components Implemented:

1. Tensor System (tensor.h):
   - Support for all data types including low-bit networks
   - Reference counting for memory management
   - Device support (CPU/GPU)
   - Shape manipulation operations

2. Automatic Differentiation (autodiff.h):
   - Variable abstraction with gradient tracking
   - Arithmetic operations with gradient computation
   - Activation functions with gradients
   - Backward pass implementation

3. Computational Graph (graph.h):
   - Directed acyclic graph structure
   - Node types: Variable, Operation, Constant, Placeholder, Output
   - Edge directions: Forward (data), Backward (gradient)
   - Graph traversal algorithms: DFS, BFS, topological sort

4. Build Systems:
   - CMake with configurable options (CUDA, ONNX, tests, examples)
   - Makefile for simple builds
   - Cross-platform support (Linux, macOS, Windows)

Code Style:
- File names: snake_case.c, snake_case.h
- Function names: snake_case()
- Type names: snake_type_t
- Constants: SNAKE_CASE
- Comments: English only, no Chinese characters
- Indentation: 4 spaces, no tabs
- Line length: 100 characters maximum
